## 常用命令

### 后端管理
```bash
pm2 start server.js --name backend    # 启动后端
pm2 status                             # 查看状态
pm2 logs backend                       # 查看日志
pm2 restart backend                    # 重启后端
pm2 stop backend                       # 停止后端
```

### Nginx 管理
```bash
nginx -t                    # 测试配置
systemctl restart nginx     # 重启 Nginx
systemctl status nginx      # 查看状态
```

### 前端构建
```bash
cd /var/www/xlgjx/frontend
npm run build               # 构建到 dist 目录
```

## Nginx 配置

配置文件位置：`/etc/nginx/sites-available/xlgjx`

```nginx
server {
    listen 80;
    server_name zz123.xyz www.zz123.xyz;

    # 前端静态文件
    location / {
        root /var/www/xlgjx/frontend/dist;
        index index.html;
        try_files $uri $uri/ /index.html;
    }

    # 后端 API 代理
    location /api {
        proxy_pass http://127.0.0.1:3000;
        proxy_http_version 1.1;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }
}
```

## 常见问题

### 1. 网站打不开
- 检查安全组是否开放 80 端口
- 检查 Nginx 是否运行：`systemctl status nginx`
- 检查后端是否运行：`pm2 status`

### 2. API 报错
- 检查后端日志：`pm2 logs backend`
- 重启后端：`pm2 restart backend`

### 3. 前端修改不生效
- 确认已执行 `npm run build`
- 清除浏览器缓存再测试

### 4. git push 失败
- 可能需要代理：`git config --global http.proxy http://127.0.0.1:8800`
- 或多试几次
